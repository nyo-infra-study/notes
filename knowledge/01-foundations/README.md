# Foundations

[← Back to Knowledge Base](../README.md)

## Overview

Essential command-line skills for working with infrastructure. This section covers the Linux tools you'll use every day — processing logs, diagnosing system issues, and moving files across machines. Master these before moving on to Kubernetes and the Argo ecosystem.

## Learning Path

1. **[Text Manipulation](./01-text-manipulation.md)** - Process logs and configuration files with sed, awk, grep, cut, sort, and uniq
2. **[System Monitoring](./02-system-monitoring.md)** - Diagnose performance and resource issues with lsof, vmstat, ps, and top
3. **[Networking Basics](./03-networking-basics.md)** - Transfer files and debug connectivity with scp and related tools

## Prerequisites

None — this is the entry point for new learners.

## What You'll Learn

- How to slice and transform text output from any command
- How to identify what processes are doing and what resources they're consuming
- How to securely copy files between machines and debug network connectivity

## Estimated Time

⏱️ 2-3 hours total

## Difficulty

🎯 Beginner
